# Vector - Реализация динамического массива на C++

## Описание

Проект представляет собой реализацию класса `Vector` - динамического массива целых чисел с функциональностью, аналогичной упрощённой версии `std::vector` из стандартной библиотеки C++.

Класс предоставляет основные операции для работы с динамическим массивом: добавление/удаление элементов, доступ по индексу, автоматическое управление памятью, а также перегруженные операторы для математических операций и вывода.

## Особенности

- **Динамическое управление памятью**: автоматическое изменение размера при необходимости
- **Безопасность**: конструкторы копирования и деструкторы для правильного управления памятью
- **Гибкость**: различные способы добавления элементов (в начало, конец, произвольную позицию)
- **Перегруженные операторы**: поддержка математических операций и удобного вывода
- **Проверки**: обработка ошибок при операциях с векторами разного размера

## Интерфейс класса

### Конструкторы

- `Vector()` - создает пустой вектор с начальной емкостью 10
- `Vector(size_t initial_size)` - создает вектор заданного размера
- `Vector(const Vector& other)` - конструктор копирования

### Основные методы

- `void push_back(int val)` - добавление элемента в конец
- `void push_front(int val)` - добавление элемента в начало
- `void insert(size_t pos, int val)` - вставка элемента в указанную позицию
- `void erase(size_t pos)` - удаление элемента по позиции
- `size_t get_size() const` - получение текущего размера
- `void resize(size_t new_capacity)` - изменение емкости внутреннего массива

### Перегруженные операторы

- `int& operator[](size_t index)` - доступ к элементу по индексу
- `const int& operator[](size_t index) const` - константный доступ
- `bool operator==(const Vector& other) const` - сравнение на равенство
- `Vector operator+(const Vector& other) const` - поэлементное сложение
- `Vector operator*(const Vector& other) const` - поэлементное умножение
- `friend std::ostream& operator<<` - вывод вектора в поток

## Пример использования

```cpp
#include "Vector.h"
#include <iostream>

int main() {
    // Создание векторов
    Vector v1;
    Vector v2(3); // вектор размера 3
    
    // Добавление элементов
    v1.push_back(1);
    v1.push_back(2);
    v1.push_back(3);
    v1.push_front(0); // добавляем в начало
    
    // Вставка и удаление
    v1.insert(2, 10); // вставляем 10 на позицию 2
    v1.erase(1);      // удаляем элемент на позиции 1
    
    // Копирование
    Vector v3 = v1;
    
    // Сравнение
    bool equal = (v1 == v3);
    
    // Математические операции
    Vector v4(4), v5(4);
    // ... инициализация векторов
    Vector sum = v4 + v5;    // поэлементное сложение
    Vector prod = v4 * v5;   // поэлементное умножение
    
    // Вывод
    std::cout << "Vector: " << v1 << std::endl;
    
    return 0;
}
```

## Ограничения

- Вектор хранит только целые числа (`int`)
- При математических операциях с векторами разного размера возвращается пустой вектор
